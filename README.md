Este es mi primer proyecto de portfolio web, desarrollado mientras estoy aprendiendo HTML5 y CSS.

El objetivo del proyecto fue practicar la estructura básica de una página web, el uso de etiquetas semánticas, estilos con CSS y la organización de contenido en secciones.

Durante su desarrollo utilicé asistencia de IA como apoyo para escribir y comprender el código, analizando cada parte para entender cómo funciona y cómo se construye una página web desde cero.

El proyecto incluye secciones como presentación personal, habilidades, proyectos y formulario de contacto, además de diseño responsive básico y buenas prácticas iniciales de SEO.

Aunque es un proyecto guiado en gran parte, representa un avance importante en mi aprendizaje y marca el inicio de mi camino como desarrollador web.
